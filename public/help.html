Setting Up AI Image Recognition

The photo import feature uses Claude's vision AI to automatically identify movies from photos of your disc cases. To enable this feature, you'll need an Anthropic API key.

Getting an API Key

1

Create an Anthropic Account

Go to console.anthropic.com and sign up for an account if you don't have one.

2

Generate an API Key

Navigate to the API Keys section in your dashboard and create a new API key. Copy it somewhere safe - you won't be able to see it again.

3

Add Credits to Your Account

Add billing information and credits to your account. The vision API costs approximately $0.01-0.03 per image analyzed.

4

Configure the App

Create a .env file in the app's root folder (next to package.json) with your API key:

ANTHROPIC_API_KEY=sk-ant-api03-your-key-here
5

Restart the Server

Stop the server (Ctrl+C) and start it again with npm start. You should see "Photo import: ENABLED" in the console.

Note: Keep your API key secret! Don't share it or commit it to version control.

Adding Movies via Photo Import

Once AI recognition is set up, you can add movies by photographing your disc cases.

Taking Good Photos

  • Capture the spines: The AI reads movie titles from the spine of cases, so make sure they're clearly visible.
  • Good lighting: Avoid glare and shadows that obscure text.
  • Steady shot: Blurry photos are harder to read.
  • Multiple photos: For large collections, take several photos of different shelf sections.

Using the Import Feature

  1. Click the "Import from Photo" button on the main page.
  2. Drag and drop a photo onto the upload area, or click to select a file.
  3. Wait while the AI analyzes the image (usually 5-15 seconds).
  4. Review the detected movies:
    • Duplicates are automatically detected and unchecked.
    • Low confidence matches are highlighted - double-check these.
    • Uncheck any movies you don't want to add.
  5. Click "Add Selected Movies" to add them to your collection.

Tip: The AI works best with clear, well-lit photos where movie spines are easily readable. If it misses some titles, try a closer photo of that section.

Automatic Folder Watching

As an alternative to the upload button, you can drop photos directly into the sources folder. The app automatically watches this folder and will process any new images.

movies/ sources/ ← Drop photos here processed/ ← Photos moved here after import

When a new photo is detected:

  1. The AI analyzes it automatically
  2. High-confidence matches (90%+) are added to your collection
  3. The photo is moved to the processed folder

Note: Folder watching adds movies automatically without review. Use the "Import from Photo" button if you want to review matches before adding.

Managing Your Collection

Marking Movies for Upgrade

To track which DVDs you want to replace with Blu-ray or 4K:

  1. Click Edit on any movie
  2. Check "Want to upgrade"
  3. Select your target format (Blu-ray or 4K)
  4. Save the changes

Use the "Wants Upgrade" filter to see all movies you want to upgrade.

Searching and Filtering

  • Search: Type in the search box to filter by title
  • Format: Filter by DVD, Blu-ray, 4K, or Mixed
  • Sort: Sort by title, format, or date added
  • View: Switch between list and grid views

Data Storage

Your movie collection is stored in a JSON file:

movies/data/movies.json

Automatic backups are created before each change in:

movies/data/backups/

You can manually edit the JSON file if needed, but be sure to restart the server afterward.
